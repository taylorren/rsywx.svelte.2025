# Images Directory

## Open Graph Image

You need to add a default Open Graph image at `og-default.jpg` for social media sharing.

**Recommended specifications:**
- Size: 1200x630 pixels
- Format: JPG or PNG  
- File size: Under 1MB
- Content: Site logo, name "任氏有无轩", or representative image

This image will be used when your site is shared on social media platforms like Facebook, Twitter, etc.

## Book Covers

Book cover images should be placed in the `/covers/` directory with the naming pattern:
- `{bookid}.webp` - for individual book covers

## Other Images

- `favicon.ico` - Site favicon
- `wechat_account.jpg` - WeChat personal QR code  
- `wechat_talk.jpg` - WeChat subscription QR code
- `tr.webp` - Fallback image for WordPress blog posts (400x200px, WebP format)
